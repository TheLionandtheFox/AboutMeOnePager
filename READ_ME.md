## Installation & Nutzung

Wenn Sie das Projekt lokal ausführen möchten, folgen Sie bitte diesen Schritten:

1. **Repository klonen:**

    ```bash
    git clone <repository-url>
    ```

2. **Verzeichnis wechseln:**

    ```bash
    cd IhrRepositoryname
    ```

3. **Abhängigkeiten installieren:**

    ```bash
    npm install 
    # oder pip install -r requirements.txt
    ```

4. **Projekt starten:**

    ```bash
    npm start
    # oder python app.py
    ```
